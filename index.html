Code ReViewer

VS Code extension to visualize code structure and analyze code quality

IntroductionMetrics reference

Seamless IDE integration

Code ReViewer works directly within your VS Code editor without the need for switching between applications.

Readable call graph

See the structure of your code at a glance, without the need to navigate through multiple files.

Large metrics library

Each function is evaluated using a wide range of metrics, providing you with a comprehensive understanding of your code's quality.

Extensibility

Provide definitions for your own metrics in Lua and extend the functionality of Code ReViewer to suit your specific needs.

Language support

Code ReViewer supports C, Java and Python, with more languages to be added in the future.

Fast code analysis

The extension can analyze about 4000 lines of code per second.
